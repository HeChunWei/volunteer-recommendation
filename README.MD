# 高考志愿填报系统
---------------------------
### 项目主要功能
1. 普通志愿填报功能
    * 选考科目
    * 位次号
    * 专业
        * 大类 (理工类)
        * 大专业 (计算机系)
        * 小专业 (软件工程)
    * 地区
    * 院校类别
    * 院校等级
    * 学费
    * 冲保稳的个数
    <br>
2. 职业生涯规划推荐
    * 通过问卷的形式
<br>
3. 院校及专业介绍
    * 数据来源: 百度百科、阳光高考
<br>
4. VIP一对一在线指导（视频通话/聊天功能）
    * webrtc
<br>
